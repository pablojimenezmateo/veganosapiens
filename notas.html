Recetas Notas Veganismo
  • Para balancear un exceso de azúcar, añade ácido (como el zumo de un limón)
  • Para balancear un exceso de ácido, añade grasa (como nata o leche de soja)
  • Para balancear un exceso de sal, añade más ingredientes o más agua
  • Para balancear un exceso de picante, añade grasa (como nata o leche de soja)
